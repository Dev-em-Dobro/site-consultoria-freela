# Site Consultoria Freela — Renda Extra com Claude Code

Landing page de vendas da consultoria **Renda Extra com Claude Code** (Dev em Dobro).

Página estática, self-contained (`index.html` + `assets/`), nas cores do Claude Code
(coral/terracotta sobre creme). Fonte editorial (Fraunces + Inter).

## Estrutura

```
index.html        # a landing page completa (CSS inline)
assets/
  cadu.png        # foto do Cadu (seção consultoria individual)
  beto.png        # foto do Beto
```

## Rodar localmente

Abra o `index.html` no navegador, ou sirva a pasta:

```bash
npx serve .
```

## Deploy

Estático — publica direto na Vercel, Netlify ou GitHub Pages (raiz = `index.html`).
